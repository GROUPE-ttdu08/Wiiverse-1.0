Les bonnes manières de Wiiverse

Voici quelques directives importantes pour faire de Wiiverse une expérience amusante et agréable pour tout le monde.

Le Code de conduite Wiiverse contient des informations détaillées, veuillez donc le lire attentivement.

Les messages peuvent être consultés dans le monde entier

Wiiverse contient de nombreuses communautés de jeu où les gens du monde entier peuvent partager leurs réflexions.

Lorsque vous publiez dans une communauté, n'oubliez pas que tout le monde peut le voir, alors s'il vous plaît, exprimez-vous d'une manière que tout le monde peut apprécier. Faites preuve de bon sens et réfléchissez avant de publier. Wiiverse est un service qui est également accessible depuis Internet, alors gardez à l'esprit que les personnes qui n'utilisent pas Wiiverse peuvent également voir vos messages.

De plus, tous les commentaires que vous ferez sur les publications de vos amis seront vus non seulement par vos amis, mais aussi par des personnes du monde entier. Veuillez garder cela à l'esprit.

Soyez gentils les uns envers les autres

Afin de garder Wiiverse un endroit amusant pour tout le monde, nous vous demandons d'être attentif aux autres utilisateurs.

Aidez-nous à faire de Wiiverse une expérience agréable en ne publiant rien d'inapproprié ou d'offensant.

Ne publiez pas d'informations personnelles—les vôtres ou celles des autres

N'oubliez pas que connaître quelqu'un dans Wiiverse n'est pas la même chose que de le connaître dans la vraie vie.

Ne partagez jamais votre adresse e-mail, votre adresse personnelle, votre nom de travail ou d'école, ou d'autres informations d'identification personnelle avec quiconque sur Wiiverse, et ne partagez jamais les informations de quelqu'un d'autre non plus.

De plus, si quelqu'un que vous rencontrez dans Wiiverse vous invite à le rencontrer dans le monde réel, n'acceptez pas. Wiiverse est une communauté en ligne et ne doit pas être utilisé pour organiser des rencontres dans le monde réel.

Ne postez pas de spoils

Certaines personnes viennent sur Wiiverse à la recherche de trucs et astuces pour les jeux, mais d'autres veulent découvrir les secrets d'un jeu par elles-mêmes.

Les messages qui révèlent les secrets d'un jeu ou de son histoire sont appelés "spoilers".

Il est possible de masquer un message "Spoil" en utilisant l'icone dans la zone pour publier un message.

Violations du code de conduite

Notre objectif est de garder Wiiverse amusant et agréable pour tout le monde.

Dans le cas où quelqu'un violerait le Code de conduite Wiiverse, nous prendrons les mesures appropriées, pouvant aller jusqu'au blocage de l'utilisateur, l'utilisateur bloquer pourras toujours aller sur Wiiverse mais il ne pourras plus publier de messages. ATTENTION : Nous pouvons aussi bannir tous votre réseau de Wiiverse (C'est-à-dire que tous les appareils connectés à votre réseau local). Tous messages publié sur Wiiverse ne peut être supprimer QUE par les administrateurs. (Même l'auteur du message ne peux pas le supprimé donc faites bien attention avant de publier un message sur Wiiverse).

Par les joueurs, à propos des jeux

Wiiverse est une communauté de jeu qui permet aux gens du monde entier d'interagir et de discuter des jeux qu'ils aiment.

Dans les communautés dédiées à un jeu spécifique, veuillez rester sur le sujet et ne publiez que des messages pertinents pour cette communauté et qui contribuent à la discussion globale. Lorsque quelqu'un publie sur un sujet sans rapport, cela peut nuire aux utilisateurs qui souhaitent discuter du jeu.

Types de violations

De plus, le contenu suivant est interdit d'être publié ou inclus dans les messages par le code de conduite Wiiverse.

  • Informations personnelles

    Les informations personnelles comprennent, mais sans s'y limiter, votre adresse e-mail, votre adresse personnelle, votre nom au travail ou à l'école et votre numéro de téléphone. N'utilisez jamais Wiiverse comme moyen de mettre en place des rencontres dans le monde réel. N'écrivez ou ne demandez jamais en public des informations de compte ou des identifiants pour d'autres services, ou toute information qui permettrait aux personnes d'être contactées directement.

  • Contenu violent

    Ce type de contenu inclut tout ce qui décrit la violence, encourage le suicide ou approuve des actes de cruauté ou de violence.

  • Inapproprié/Nocif

    Le contenu inapproprié ou préjudiciable comprend tout ce qui encourage un comportement dangereux ou des activités illégales.

  • Haineux/Intimidation

    Cela inclut tout contenu calomniant, diffamant ou déformant une autre personne, ainsi que tout contenu discriminatoire, harcelant ou abusif.

  • Sexuellement explicite

    Le contenu sexuellement explicite comprend tout ce qui contient de la nudité, de la sexualité ou des propositions.

  • Signalement inapproprié

    Signaler intentionnellement une publication erronée pour violation du code de conduite Wiiverse est en soi une violation du code de conduite Wiiverse.

  • Harcèlement, insultes en vers une personne
  • Wiiverse est contre le harcèlement, l'harcèlement sur Wiiverse est une violation du Code de Conduite Wiiverse.
  • Création de ses propres communauté sans l'autorisation d'un administrateur
  • Créer ses communauté sans l'autorisation viole le Code de Conduite Wiiverse.

Chaque utilisateur Wiiverse est responsable de comprendre et de se conformer au Code de conduite Wiiverse.

Une page de signalement seras bientôt disponible à l'acceuil.

Nous pouvons mettre à jour le Code de conduite Wiiverse à tout moment, surtout au moment des mises à jours.
